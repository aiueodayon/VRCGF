## 🚀 なぜ VRCGF なのか？

VRChatの世界は無限大ですが、**「今夜、どこに行こう？」**と迷う時間も増えていませんか？

**VRCGF** は、単なる検索ツールではありません。あなたの好みに合ったイベントやコミュニティを「発見」し、そこへ「繋がる」までの体験をデザインした**コンパニオンツール**です。

---

## ✨ 視覚で楽しむ「ディスカバリー」

文字だけのリストから探すのはもう終わりです。雑誌をめくるような感覚で、新しいイベントやワールドに出会いましょう。

* **マガジン風レイアウト**: 注目のイベントを美しいカードで表示。
* **直感的な雰囲気確認**: タイトルだけでなく、サムネイルの空気感で探せます。

![Discovery UI Screenshot](https://i.gyazo.com/93dc56c9c1392e15e98f836512f7540e.png)

---

## 📝 「保存」より気楽な「メモ」機能

「お気に入り登録」だと、絶対に行かなきゃいけない気がしませんか？
VRCGFは、もっと直感的な **「メモ（Memo）」** を採用しました。

* **ワンクリックでキープ**: 「あとで見るかも」「ちょっと気になる」レベルでOK。
* **ON / OFF が簡単**: メモるのも、外すのも一瞬。思考を邪魔しません。

![Memo Feature Screenshot](https://i.gyazo.com/e3cdafbca77b79a769544cb5e083ca5a.png)

---

## 🔍 欲しい情報に「1秒」で到達

膨大なイベントの中から、今の気分にピッタリなものだけを抽出します。

* **リアルタイムフィルタ**: 入力と同時に結果を表示。
* **スマート条件**: 「今夜開催」「PCのみ」「音楽イベント」など、詳細な絞り込みが可能。
* **ステータスバッジ**: 「開催中」「あと30分」が一目でわかります。

![Search and Filter Screenshot](https://i.gyazo.com/bd054e2e2ef2b8ac2c3ca4539ccc6b77.png)

---

## 🤝 つながりを広げる機能

* **X (Twitter) 連携**: 「これ面白そう！」を友達に即シェア。
* **Googleカレンダー追加**: 行きたいイベントを逃さないよう、スケジュールへワンクリック登録。

---

## 🛡️ 安心・安全へのこだわり

VRCGFは、あなたのプライバシーとアカウントの安全を最優先に設計されています。

| 項目 | VRCGFの方針 |
| :--- | :--- |
| **ログイン情報** | ❌ **一切保持しません**（ローカル完結） |
| **データ保存** | ✅ あなたのPC内にのみ保存されます |
| **トラッキング** | ❌ 行動監視・追跡は行いません |
| **規約遵守** | ✅ VRChat公式APIを正当な方法でのみ利用 |

> **Note:** クライアントの改変（Mod）は一切行いません。安心してご利用ください。
